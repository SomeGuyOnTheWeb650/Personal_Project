Project Description
---------------------
Basic bank statement summarizer, should create a file that contains simplified expenses and deposits for a set time period
Does not currently automatically pull data from anywhere, need to manually move pdf into data, 
e.g. april 2026 -> 202604 so that the files are in ascending order

Note to prospective users, no real functionality beyond my own use, didn't find a clean way to convert pdf to text, in a standard way across banks, within the time-frame I alloted
GUI implemented was more of a tinker toy, learning the absolute basics of tkinter, looks ugly, didn't want to spend excessive time refining my approach and moving on to other projects
